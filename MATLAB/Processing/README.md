# MATLAB/Processing

Logical grouping for MATLAB processing code.

Key scripts:
- `run_matlab.m` (single entry point, located in repo root)
- `main_run_sim.m` (momentum-only pipeline)
- `main_run_full_sim.m` (full attitude + wheels pipeline)
- `simulate_mtq.m`, `simulate_mtq_full.m`
- Shared modules in `models/`, `controllers/`, `plotting/` (within this folder)
